A plugin that logs user's recent post views and allows users to save their favorite posts

== Description ==

This plugin automatically logs viewed posts and products (if WooCommerce is enabled) of logged in users to your Wordpress database. Optionally, 
you can display a button via shortcode to allow visitors to save a post or product to a list. 

Both Recently viewed and Saved Posts can be displayed to the logged-in user on any page such as the WooCommerce dashboard. 

Shortcodes available
--------------------------------
[recentposts] - Shows all recently visited posts.
[savedposts] - Shows all saved posts.
[savepost-button] - Displays the button to save the post

== Installation ==

1. Download and unzip the plugin.
2. Upload `recent-saved-posts` to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in Wordpress.
4. Add shortcodes where necessary

== Configuration ==

There are currently no settings available for this plugin.
